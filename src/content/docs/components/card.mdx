# Card

The Card component is a flexible container that groups related content and actions. It supports headers, footers, and various content layouts.

## Import

```tsx
import { Card } from 'betterui'
```

## Usage

### Basic Card

```tsx
<Card>
  <Card.Header>
    <Card.Title>Card Title</Card.Title>
    <Card.Description>Optional card description</Card.Description>
  </Card.Header>
  <Card.Content>
    Main content goes here
  </Card.Content>
  <Card.Footer>
    Footer content
  </Card.Footer>
</Card>
```

### Simple Content Card

```tsx
<Card>
  <Card.Content>
    A simple card with only content
  </Card.Content>
</Card>
```

### With Image

```tsx
<Card>
  <Card.Image
    src="/placeholder.jpg"
    alt="Card image"
    width={400}
    height={200}
  />
  <Card.Header>
    <Card.Title>Featured Image</Card.Title>
  </Card.Header>
  <Card.Content>
    Content below the image
  </Card.Content>
</Card>
```

### Interactive Card

```tsx
<Card className="hover:shadow-lg transition-shadow">
  <Card.Header>
    <Card.Title>Interactive Card</Card.Title>
  </Card.Header>
  <Card.Content>
    This card has hover effects
  </Card.Content>
  <Card.Footer>
    <Button variant="outline">Learn More</Button>
  </Card.Footer>
</Card>
```

## Props

### Card

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |
| `children` | `ReactNode` | - | Card content |

### Card.Header

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |
| `children` | `ReactNode` | - | Header content |

### Card.Title

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |
| `children` | `ReactNode` | - | Title content |
| `as` | `'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6'` | `'h3'` | HTML heading element to use |

### Card.Description

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |
| `children` | `ReactNode` | - | Description content |

### Card.Content

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |
| `children` | `ReactNode` | - | Main content |

### Card.Footer

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |
| `children` | `ReactNode` | - | Footer content |

### Card.Image

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image source URL |
| `alt` | `string` | - | Image alt text |
| `width` | `number` | - | Image width |
| `height` | `number` | - | Image height |
| `className` | `string` | `''` | Additional CSS classes |

## Customization

### Custom Styles

You can customize the card's appearance using Tailwind classes:

```tsx
<Card className="bg-gradient-to-br from-purple-50 to-pink-50 dark:from-purple-900/20 dark:to-pink-900/20">
  <Card.Header>
    <Card.Title className="text-gradient bg-clip-text text-transparent bg-gradient-to-r from-purple-500 to-pink-500">
      Gradient Card
    </Card.Title>
  </Card.Header>
  <Card.Content>
    A card with gradient background and title
  </Card.Content>
</Card>
```

### Theme Customization

To customize the default styles for all cards, modify your theme configuration:

```typescript
// tailwind.config.ts
export default {
  theme: {
    extend: {
      borderRadius: {
        card: '0.75rem',
      },
      spacing: {
        'card-padding': '1.5rem',
      },
    },
  },
}
```
