# Getting Started with Better UI

Welcome to Better UI, a modern, customizable UI component library built with Next.js and Tailwind CSS.

## Installation

```bash
pnpm add betterui
```

## Basic Usage

```tsx
import { Button } from 'betterui'

export default function MyComponent() {
  return (
    <Button variant="primary">
      Click me
    </Button>
  )
}
```

## Key Features

- 🎨 Modern design system
- 🔧 Highly customizable components
- ⚡️ Built for performance
- 📱 Responsive by default
- ♿️ Accessible components
