# Radio

The Radio component allows users to select a single option from a list of choices. It's designed to be accessible and customizable.

## Import

```tsx
import { Radio, RadioGroup } from 'betterui'
```

## Usage

### Basic Radio Group

```tsx
<RadioGroup defaultValue="option1">
  <div className="space-y-2">
    <div className="flex items-center space-x-2">
      <Radio id="option1" value="option1" />
      <Label htmlFor="option1">Option 1</Label>
    </div>
    <div className="flex items-center space-x-2">
      <Radio id="option2" value="option2" />
      <Label htmlFor="option2">Option 2</Label>
    </div>
    <div className="flex items-center space-x-2">
      <Radio id="option3" value="option3" />
      <Label htmlFor="option3">Option 3</Label>
    </div>
  </div>
</RadioGroup>
```

## Props

### RadioGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The controlled value of the radio group |
| `defaultValue` | `string` | - | The default value of the radio group |
| `onValueChange` | `(value: string) => void` | - | Called when the value changes |
| `name` | `string` | - | The name of the radio group |
| `required` | `boolean` | `false` | Whether the radio group is required |
| `disabled` | `boolean` | `false` | Whether the entire group is disabled |
| `className` | `string` | `''` | Additional CSS classes |

### Radio

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The value of the radio button |
| `checked` | `boolean` | - | Whether the radio is checked |
| `defaultChecked` | `boolean` | `false` | The default checked state |
| `disabled` | `boolean` | `false` | Whether the radio is disabled |
| `required` | `boolean` | `false` | Whether the radio is required |
| `name` | `string` | - | The name of the radio button |
| `id` | `string` | - | The ID of the radio button |
| `className` | `string` | `''` | Additional CSS classes |

## Examples

### Controlled Radio Group

```tsx
function ControlledRadioGroup() {
  const [value, setValue] = useState('option1')

  return (
    <RadioGroup value={value} onValueChange={setValue}>
      <div className="space-y-2">
        <div className="flex items-center space-x-2">
          <Radio id="option1" value="option1" />
          <Label htmlFor="option1">Option 1</Label>

          <Radio id="option2" value="option2" />
          <Label htmlFor="option2">Option 2</Label>
        </div>

  )
}
```

## Accessibility

The Radio component follows WAI-ARIA guidelines:

- Uses native `<input type="radio">` elements
- Supports keyboard navigation (Tab, Space)
- Groups related radios with `role="radiogroup"`
- Associates labels using `htmlFor`
- Includes proper ARIA attributes
