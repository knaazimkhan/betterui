## Overview

The `Aurora Background` creates a smooth, animated aurora effect with multiple flowing waves. This component is built using `Framer Motion` for seamless animations and is designed to enhance modern UI designs.

It is highly customizable, allowing you to adjust the number of waves, colors, and animation speed. The aurora effect is created using a combination of linear gradients and motion animations.
