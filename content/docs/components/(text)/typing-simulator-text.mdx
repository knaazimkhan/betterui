## Overview

The `TypingSimulatorText` component is designed to simulate typing effects for a text or a sequence of texts. It provides an animated text display where the characters are typed and then deleted in a looping or one-time animation cycle.

The component also offers flexibility in customizing the typing and deleting speeds, as well as the appearance of the cursor. You can use it for creating dynamic and engaging user interfaces where text is dynamically typed out.

## Examples

### text

This prop defines the text that will be typed. It can either be a string or an array of strings.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text={["Hello", "World!"]} />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { TypingSimulatorText } from '@/components/text/typing-simulator-text'

    <TypingSimulatorText text="Hello World!" />
    
    Or use an array of strings to type multiple messages:

    <TypingSimulatorText text={["Hello", "World!"]} />
  ```
  </Tab>
</Tabs>

### typingSpeed

The typingSpeed prop controls how fast the text is typed out. The default value is 100.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text="Fast typing speed" typingSpeed={50} />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { TypingSimulatorText } from '@/components/text/typing-simulator-text'

    <TypingSimulatorText text="Fast typing speed" typingSpeed={50} />
  ```
  </Tab>
</Tabs>
This example will type the text faster.

### deletingSpeed

The deletingSpeed prop controls how fast the text is deleted. The default value is 50.

  <Tab value="Preview">

    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx

    <TypingSimulatorText text="Slow deleting speed" deletingSpeed={200} loop={true} />
  ```
  </Tab>
</Tabs>
This example will delete the text slowly.

### pauseTime

The pauseTime prop controls how long the text will pause before deleting. The default value is 1500.

  </Tab>
</Tabs>
This example will wait for 3 seconds before starting to delete the text.

  </Tab>
