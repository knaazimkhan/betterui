## Examples

### text

This prop defines the text that will be typed. It can either be a string or an array of strings.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text={["Hello", "World!"]} />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { GradientWaveText } from '@/components/text/gradient-wave-text'

    <TypingSimulatorText text="Hello World!" />
    
    Or use an array of strings to type multiple messages:

    <TypingSimulatorText text={["Hello", "World!"]} />
  ```
  </Tab>
</Tabs>

### typingSpeed

The typingSpeed prop controls how fast the text is typed out. The default value is 100.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text="Fast typing speed" typingSpeed={50} />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { GradientWaveText } from '@/components/text/gradient-wave-text'

    <TypingSimulatorText text="Fast typing speed" typingSpeed={50} />
  ```
  </Tab>
</Tabs>
This example will type the text faster.

### deletingSpeed

The deletingSpeed prop controls how fast the text is deleted. The default value is 50.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text="Slow deleting speed" deletingSpeed={200} loop={true} />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { GradientWaveText } from '@/components/text/gradient-wave-text'

    <TypingSimulatorText text="Slow deleting speed" deletingSpeed={200} loop={true} />
  ```
  </Tab>
</Tabs>
This example will delete the text slowly.

### pauseTime

The pauseTime prop controls how long the text will pause before deleting. The default value is 1500.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text="Pause before deleting" pauseTime={3000} loop={true} />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { GradientWaveText } from '@/components/text/gradient-wave-text'

    <TypingSimulatorText text="Pause before deleting" pauseTime={3000} loop={true} />
  ```
  </Tab>
</Tabs>
This example will wait for 3 seconds before starting to delete the text.

### cursorStyle

The cursorStyle prop defines the character or style of the cursor. By default, it is a pipe (|).

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text="Different cursor" cursorStyle="_" />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { GradientWaveText } from '@/components/text/gradient-wave-text'

    <TypingSimulatorText text="Different cursor" cursorStyle="_" />
  ```
  </Tab>
</Tabs>
This example will change the cursor style to any custom character.

### loop

The loop prop, when set to true, will restart the typing animation after it completes.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text="Looping text" loop={true} />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { GradientWaveText } from '@/components/text/gradient-wave-text'

    <TypingSimulatorText text="Looping text" loop={true} />
  ```
  </Tab>
</Tabs>
This will make the typing effect restart after completion.

### className

The className prop allows you to add custom styling to the component.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Preview">
    <PreviewComponent>
      <TypingSimulatorText text="Custom styling" className="text-yellow-500" />
    </PreviewComponent>
  </Tab>
  <Tab value="Code">
  ```tsx
    'use client'

    import { GradientWaveText } from '@/components/text/gradient-wave-text'

    <TypingSimulatorText text="Custom styling" className="text-yellow-500" />
  ```
  </Tab>
</Tabs>
This will apply text-4xl and font-bold classes to the component.
